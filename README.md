# Map explorer

This repo is a very simple project that plays around with CSS transformations and animations.

The main goal was to get some experience with [Parcel](https://parceljs.org/) bundler. Code is written in [TypeScript](https://www.typescriptlang.org/), CSS with [SASS](https://sass-lang.com/) and bundled all together with [Parcel](https://parceljs.org/).
